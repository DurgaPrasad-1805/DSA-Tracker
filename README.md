# DSA Tracker – 90 Day Coding Journey

A full-stack web application designed to track my daily Data Structures and Algorithms (DSA) practice through a structured **90-day roadmap**.

The platform allows me to monitor my learning progress, track solved problems, maintain streaks, and visualize performance using interactive charts.

---

## 🌐 Live Demo

Try the application here:

https://dsa-tracker-cuee.onrender.com

---

# 📌 Features

## Progress Tracking

Track solved DSA problems across multiple topics including:

* Arrays
* Strings
* Linked Lists
* Trees
* Graphs
* Dynamic Programming
* Greedy Algorithms

---

## 90-Day Structured Roadmap

The learning journey is divided into **5 phases**:

1. **Phase 1 – Foundations**

   * Algorithmic Thinking
   * Arrays
   * Binary Search
   * Strings
   * Linked Lists

2. **Phase 2 – Core Data Structures**

   * Stack
   * Queue
   * Trees
   * Binary Search Trees

3. **Phase 3 – Problem Solving Patterns**

   * Sliding Window
   * Two Pointers
   * Fast & Slow Pointers
   * Merge Intervals
   * Cyclic Sort

4. **Phase 4 – Trees & Graphs**

   * BFS / DFS
   * Topological Sort
   * Shortest Path Algorithms

5. **Phase 5 – Advanced Algorithms**

   * Dynamic Programming
   * Greedy Algorithms

---

## Streak Tracker

Motivates consistent practice by tracking:

* Current coding streak
* Longest streak

---

## Weekly Progress Graph

Visualizes problem-solving progress using **Chart.js**.

Example:

Week 1 → 18 problems
Week 2 → 22 problems
Week 3 → 27 problems

# How to Run Locally

Clone the repository:

```
git clone https://github.com/DurgaPrasad-1805/DSA-Tracker.git
```

Move into the project folder:

```
cd DSA-Tracker
```

Install dependencies:

```
pip install flask gunicorn
```

Run the application:

```
python3 app.py
```

Open in browser:

```
http://localhost:5000
```
